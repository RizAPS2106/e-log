# Aplikasi E-Log

Aplikasi berbasis website untuk mengelola data poin perilaku siswa juga data pembimbing, orang tua, kurikulum, bk dsb. Dibangun menggunakan bahasa php. 

## Tech Stack
- PHP Native 
- MySQL

## Fitur Utama
- Mengelola user 'Admin', 'Pembimbing', 'Wali Kelas' dan 'Siswa'.
- Mengelola Data Jenis Pelanggaran.
- Mengelola Data Pelanggaran Siswa.
- Menampilkan Log Pelanggaran Siswa.
- Skor Siswa berdasarkan Log Pelanggaran.
